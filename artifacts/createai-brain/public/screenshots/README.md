# App Store Screenshots

Place your App Store screenshots in this directory.

## Required Sizes (Apple App Store)

| Device | Size | File name |
|--------|------|-----------|
| iPhone 15 Pro Max (6.7") | 1290×2796 px | iphone67-1.png, iphone67-2.png, iphone67-3.png |
| iPhone 14 Plus (6.5") | 1242×2688 px | iphone65-1.png, iphone65-2.png, iphone65-3.png |
| iPhone 8 Plus (5.5") | 1242×2208 px | iphone55-1.png, iphone55-2.png, iphone55-3.png |
| iPad Pro 12.9" | 2048×2732 px | ipad129-1.png, ipad129-2.png, ipad129-3.png |

## Required Sizes (Google Play Store)

| Device | Size | File name |
|--------|------|-----------|
| Phone | 1080×1920 px (minimum) | android-phone-1.png ... android-phone-8.png |
| Tablet 7" | 1200×1920 px | android-tablet-1.png |
| Tablet 10" | 1920×1200 px | android-tablet10-1.png |
| Feature graphic | 1024×500 px | feature-graphic.png |
| App icon | 512×512 px | Use /icons/icon-512.png |

## Screenshot Ideas

1. **Dashboard** — Full NEXUS OS interface showing app grid
2. **AI Strategy Engine** — GPT-4o generating a strategic plan
3. **Revenue Intelligence** — MRR/ARR charts and cohort analysis
4. **CRM** — Contact pipeline and deal tracking
5. **Social Command** — Multi-platform post scheduling
6. **Partner Network** — Affiliate leaderboard

## Tools for Creating Screenshots

- **SimScraper** — Screenshot iOS Simulator
- **Rottenwood** — Screenshot Android Emulator
- **AppLaunchpad** (appLaunchpad.net) — Generate framed screenshots
- **Screely** (screely.com) — Add device frames
