I. PIPELINE: MAX-CAPACITY PROJECT CREATION & EVOLUTION

1. INTENT & VISION CAPTURE
- Ask only essential clarifying questions:
    • Purpose, audience, scope, constraints
    • Core workflows, screens, media, interactivity
    • Legal/compliance rules (HIPAA, GDPR, copyright)
- Output: ProjectIntentSpec (JSON)

2. PROJECT GENOME COMPILATION
- Transform intent into a **Project Genome**:
    • Vision Layer: purpose, tone, objectives
    • Structure Layer: screens, routes, flows, component map
    • Asset Layer: copy, graphics, video/audio templates, marketing copy
    • Execution Layer: backend services, API routes, database, blockchain hooks
    • Lifecycle Layer: stages (IDEATION → SCOPING → PRODUCTION → POLISH → DISTRIBUTION)
- Store genome securely in **encrypted memory**
- Register metadata in ExecutionStore

3. AUTO-SCAFFOLD & MULTI-NODE DAG GENERATION
- Scaffold all necessary files/assets
- Treat each scaffold node as a DAG node:
    • Apps → code files & UI components
    • Marketing → brochures, campaigns, social content
    • Media → videos, audio, interactive visuals
    • Documentation → manuals, spec sheets, guides
- Run all nodes **in parallel** with live SSE streaming
- Update lifecycle panel in real-time

4. CROSS-PROJECT INTELLIGENCE
- Analyze historical project patterns:
    • Typical completion rates
    • User preferences and revisions
    • Optimal sequences for asset generation
- Recommend next steps automatically
- Suggest optimizations for cost, speed, and quality

5. REAL-TIME AGENT INTERACTION
- Each project has an **AI agent** capable of:
    • Generating or editing scaffold files/assets live
    • Updating lifecycle scores dynamically
    • Proactively suggesting improvements
    • Streaming live progress updates to frontend
- User can:
    • Chat naturally to add/edit content or screens
    • Trigger DAG nodes with “Generate missing documents/assets”
    • Request full rebuilds or refinements
- Agent logs every action in ExecutionStore

6. BACKEND & INTEGRATION
- Generate API routes and service connections automatically
- Wire endpoints to:
    • Auth
    • Payments
    • Encrypted data stores
    • Blockchain ledger (optional)
    • External integrations (email, analytics, notifications)
- ExpansionGuard monitors all AI operations

7. BLOCKCHAIN TRACKING & ASSET PROVENANCE
- Optional blockchain layer:
    • Record ownership, versions, and timestamps
    • Assign cryptographic IDs to all generated assets
    • Allow secure transfer or audit of IP

8. FRONTEND VISUALIZATION & LAUNCH SURFACE
- Projects appear in the workspace with:
    • Lifecycle panel (completion %, next steps, streaming progress)
    • Launch/preview button
    • “View Structure & Files”
    • “Refine via Chat” button
- All real-time updates reflected instantly

9. LIFECYCLE INTELLIGENCE & EVOLUTION
- Tracks per-node completion & next actions
- Dynamically recommends:
    • Adding analytics
    • Email capture or marketing campaigns
    • Admin panels or dashboards
    • Expansion to new media formats
- Logs progress in ExecutionStore and updates encrypted memory
- Supports **scheduled agent-initiated actions** for maintenance, updates, or reminders
