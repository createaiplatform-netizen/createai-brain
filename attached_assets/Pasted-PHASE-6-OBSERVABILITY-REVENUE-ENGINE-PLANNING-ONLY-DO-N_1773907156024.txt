FILES TO CREATE (4)
━━━━━━━━━━━━━━━━━━━━━━━

1. artifacts/api-server/src/observability/ExecutionLogger.ts
   - Handles lifecycle events:
       • start
       • success
       • error
       • blocked
   - Produces structured execution records
   - Accepts executionId, requestId, engineId, trace, tokenUsage, timestamps

2. artifacts/api-server/src/observability/CostCalculator.ts
   - Calculates cost using:
       • token usage
       • execution duration
       • optional engine multipliers
   - Configurable rates
   - Returns costEstimate

3. artifacts/api-server/src/observability/AnomalyDetector.ts
   - Detects:
       • recursion patterns
       • abnormal token spikes
       • repeated failures
   - Returns:
       {
         riskScore: number,
         flags: string[]
       }

4. artifacts/api-server/src/observability/ExecutionStore.ts
   - In-memory store for execution records
   - Supports:
       • save(record)
       • list()
       • getByExecutionId()
   - No database integration yet

EXECUTION RECORD STRUCTURE

  executionId: string
  requestId: string

  engineId: string
  trace: string[]

  depth: number

  efficiencyScore?: number
  riskScore?: number
